# Video Folder for EduVault

Place your MP4 video files in this folder for dynamic background effects.

## Required Video:
- **background.mp4** - This will be used as the dynamic background for the dashboard

## Video Specifications:
- Format: MP4
- Recommended resolution: 1920x1080 or higher
- Duration: Loop-friendly (seamless loop recommended)
- File size: Keep under 50MB for optimal performance
- Content: Educational, abstract, or nature-themed videos work best

## How to add videos:
1. Copy your MP4 video file to this folder
2. Name it `background.mp4` for the main background
3. Refresh your browser to see the changes

## Current Status:
⚠️ **No video file present** - The dashboard is currently using an animated gradient background with moving blob effects as a fallback.

## Note:
- If no video is present, the dashboard automatically uses a beautiful animated gradient background
- The animated background features smooth color transitions and floating blob effects
- This provides a dynamic, video-like experience without requiring actual video files
